# Strategic-Business-Analytics-Financial-Stability-in-Azerbaijan-Key-Industries
This project focuses on analyzing the financial performance and audit risks of four major sectors within the Azerbaijan economy: Industry (Sənaye), Construction (Tikinti), Transportation (Nəqliyyat), and ICT (İKT).

Using a combination of SQL, Excel, and Power BI, I processed financial data to evaluate profitability and stability metrics. This analysis provides a deep dive into how Azerbaijan's industrial landscape manages risks and trade cycles in the current economic climate.This project presents an end-to-end data analytics solution designed to evaluate the financial health, operational efficiency, and audit-related risks within the primary economic sectors of Azerbaijan. In a rapidly evolving economic landscape, understanding the balance between profitability and loss is crucial for strategic decision-making and long-term stability.

The analysis focuses on four pivotal sectors that drive the national economy: Industry (Sənaye), Construction (Tikinti), Transportation (Nəqliyyat), and ICT (İKT). By integrating data from SQL databases, refining it through Excel, and visualizing it via an interactive Power BI dashboard, this project transforms raw financial figures into actionable strategic insights. The core objective is to identify which sectors demonstrate robust financial "resilience" and which ones exhibit "red flags" that require immediate auditing or operational restructuring.
